Mixed Reality Color Blind Assistant

Unity Multi Player Computer Vision Accessibility

Project Overview

This project introduces an innovative Augmented and Virtual Reality (AR/VR) based assistant designed to help individuals with color vision deficiency (CVD). Developed as part of the ICCTARVR project in collaboration with ETH Zürich, this solution focuses on enhancing accessibility in digital and real-world environments for CVD users.

The system utilizes real-time image processing and computer vision algorithms to identify and modify colors, presenting them in a way that is distinguishable for users with various types of color blindness. The multiplayer component allows for collaborative sessions, where a non-impaired user can guide a CVD user in color-related tasks.

Key Features

  • Real-time color transformation for deuteranopia, protanopia, and tritanopia.
  • Multi-user collaboration mode for guided assistance.
  • Integration with Unity for cross-platform AR/VR deployment.
  • User-centric design focusing on ease of use and minimal cognitive load.

Project Info

Collaboration: ETH Zürich

Field: XR Accessibility

Status: Research Prototype
