KoCo App - Cultural Bridge

A 3D desktop app with CNN-based facial emotion recognition for foreigners in Korea, aiding cultural adaptation.

Unity 3D CNN Python API Emotion Detection
View Details ↓

Project Overview

The KoCo (Korea & Companion) application is a 3D desktop tool designed to bridge the cultural gap for foreigners residing in South Korea. It utilizes a Convolutional Neural Network (CNN) for real-time facial emotion recognition, helping users understand and respond to non-verbal cues in a Korean context.

Key Features

  • CNN-based facial emotion recognition via Python API.
  • Interactive 3D scenarios for cultural learning.
  • Aims to reduce cultural misunderstandings and improve social integration.

Project Overview

Presented at the World IT Show Coex 2024, the Virtual Doctor Assistant is a proof-of-concept VR application for healthcare. It provides a safe and immersive environment for patients to interact with a virtual medical professional for consultations, training, and therapy.

Key Features

  • Realistic virtual environment for patient-doctor interaction.
  • Incorporates emotion detection to gauge patient comfort and response.
  • Designed for applications in medical training, patient education, and tele-health.
