AR Fire Safety Prototype

Unity AR AR Foundation Mobile

About The Project

A prototype mobile application using Augmented Reality to simulate a fire emergency. The app's purpose is to augment a fire in the real world and then allow the user to extinguish it with a virtual fire extinguisher, providing an interactive training experience.
