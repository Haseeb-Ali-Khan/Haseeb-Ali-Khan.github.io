KoCo – Emotion-Aware Korean Communication 3D App

Unity 3D CNN Python API Emotion Detection Emotion Induction

About The Project

A 3D desktop application designed for foreigners in Korea to practice Korean language with a virtual character. The system uses both facial and text-based emotion recognition to detect the user’s emotional state during conversations. If negative emotions are detected, emotion induction techniques are applied to enhance the user's mood. A custom CNN model was trained for facial emotion recognition, deployed via a Dockerized API, and integrated into the application for real-time inference.

My Responsibilities

  • Designed and developed the full 3D desktop application in Unity, including UI/UX elements and character interaction systems.
  • Built conversational interface logic for the virtual Korean character to interact with users in Korean.
  • Trained a CNN model for real-time facial emotion recognition using labeled facial datasets.
  • Developed and Dockerized a RESTful API to serve the trained CNN model.
  • Integrated the API with the Unity application for live emotion-based responses during conversations.
  • Implemented emotion induction techniques to support users experiencing negative emotional states.
