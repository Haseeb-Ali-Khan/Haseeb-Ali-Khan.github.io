Mixed Reality Color Blind Assistant

Unity C# XR Color Blindness Accessibility

About The Project

An AR/VR mixed reality application designed to help colorblind users identify and interact with colored objects during real-world or virtual tasks. The system supports user-system interaction and can be extended to multiplayer mode, enabling a guide to assist the user in real time within the immersive environment.

My Responsibilities

  • Developed a Mixed Reality system in Unity to support colorblind users in object identification tasks.
  • Created interactive AR/VR environments for testing color differentiation tasks.
  • Designed user-system interaction flow and implemented task-guidance features.
  • Built multiplayer functionality enabling guide-assisted sessions for users.
  • Evaluated usability and accessibility in both AR and VR contexts.
